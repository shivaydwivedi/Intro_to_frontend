# Learning Path

This repository is designed to be completed in order. Each project introduces new concepts while repeating earlier skills.

## How to Use the Path

1. Read the project README.
2. Review the learning objectives.
3. Build the project in the `starter/` folder.
4. Test your work with the project checklist.
5. Compare your work with the `solution/` folder.
6. Write down what you learned before moving on.

## Skill Progression

### Projects 01-02: HTML and CSS Foundations

Focus on semantic HTML, readable content structure, selectors, spacing, typography, color, and simple responsive decisions.

### Projects 03-04: Layout

Practice Flexbox, CSS Grid, reusable sections, and responsive layout patterns.

### Projects 05-07: JavaScript Interaction

Use variables, functions, events, conditionals, arrays, objects, and DOM updates to make pages interactive.

### Projects 08-10: Browser APIs and Async Data

Work with forms, validation, localStorage, Fetch API, loading states, empty states, and error states.

## Learning Tips

- Build first, then compare with the solution.
- Keep changes small and test often.
- Use browser developer tools to inspect layout and debug JavaScript.
- Write code that a beginner can explain out loud.
- Revisit earlier projects when a concept feels unclear.
