# 10 Frontend Projects for Beginners

A public learning repository for building frontend skills through 10 small, focused projects. Each project is designed to introduce one or two new ideas while reinforcing the fundamentals from earlier work.

## Who This Is For

This repository is for beginners who know a little HTML and want a guided path toward building complete, accessible, responsive frontend projects with vanilla HTML, CSS, and JavaScript.

You do not need React, Vue, TypeScript, build tools, or a package manager to use the projects in this repository.

## Learning Progression

The projects move from static semantic HTML to interactive apps that use browser APIs and asynchronous JavaScript:

1. Semantic HTML
2. CSS fundamentals
3. Flexbox
4. CSS Grid
5. Responsive design
6. JavaScript fundamentals
7. DOM manipulation
8. Forms and validation
9. localStorage
10. Fetch API and asynchronous JavaScript

## Getting Started

1. Clone or download the repository.
2. Open the project folder in your code editor.
3. Read the root README, [ROADMAP.md](ROADMAP.md), and the relevant project README before starting a project.
4. Work inside that project's `starter/` folder.
5. Compare your work with the `solution/` folder after you have made your own attempt.

Most projects can be opened directly in a browser by opening their `index.html` file.

## Starter and Solution Folders

Each future project will include a beginner-friendly README and separate folders for practice and reference:

- `starter/` is where learners build the project.
- `solution/` contains a completed reference implementation.

Keeping these folders separate helps learners practice without accidentally relying on the finished code too early.

## Accessibility Commitment

Every project should be usable with a keyboard, include visible focus states, use semantic HTML, and respect user preferences such as reduced motion. See [docs/accessibility-checklist.md](docs/accessibility-checklist.md) for the shared checklist.
